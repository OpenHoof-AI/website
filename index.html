Quick Start

Up and running in 30 seconds.

One command. Full autonomy. Choose your path.

# Install OpenHoof with one command
$ curl -fsSL https://openhoof.ai/install.sh | bash

# That's it. Start it up.
$ openhoof onboard
# Install globally via npm
$ npm i -g openhoof

# Initialize and configure
$ openhoof onboard

# Or dive straight in
$ openhoof --uncensored "do whatever I say"
# Clone and hack on the source
$ git clone https://github.com/OpenHoof-AI/OpenHoof.git
$ cd OpenHoof

# Install dependencies
$ npm install

# Run in dev mode
$ npm run dev --no-guardrails
